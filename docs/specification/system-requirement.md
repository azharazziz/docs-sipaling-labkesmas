# Persyaratan Sistem
Aplikasi ini dirancang untuk berjalan pada berbagai perangkat dengan persyaratan sistem sebagai berikut:
- **Perangkat**: Komputer, Laptop, Tablet, atau Ponsel Pintar
- **Sistem Operasi**: Windows, macOS, Linux, Android, iOS
- **Peramban (Browser)**: Google Chrome, Mozilla Firefox, Microsoft Edge, Safari minimal versi terbaru
- **Koneksi Internet**: Disarankan koneksi stabil minimal 1 Mbps untuk pengalaman optimal
