# Hak Akses dan Peran Pengguna
Aplikasi ini memiliki beberapa peran pengguna dengan hak akses yang berbeda, antara lain:
| **Nama Peran**              | **Deskripsi Umum**                                                   | **Hak Akses** |
|-----------------------------|-----------------------------------------------------------------------|---------------|
| **Super Admin**             | Pengelola tertinggi sistem aplikasi.                                 | • Mengelola seluruh data dan konfigurasi sistem.<br/>• Membuat, mengubah, dan menghapus akun pengguna.<br/>• Mengatur peran (role) dan izin (permission).<br/>• Mengelola semua modul seperti SK, prosedur, registrasi klinik/lab, FAQ, dan kehadiran. |
| **Archivist**               | Petugas pengarsipan dokumen dan SK.                                  | • Melihat, menambah, mengubah, dan menghapus dokumen SK (decrees).<br/>• Memastikan data arsip tetap mutakhir dan terdokumentasi. |
| **Human Resource**          | Petugas administrasi kepegawaian.                                    | • Mengelola data pegawai dan akun pengguna.<br/>• Melihat dan memperbarui dokumen SK dan prosedur kerja.<br/>• Membuat dan memantau tautan internal. |
| **Employee (default)**      | Pegawai internal yang menggunakan sistem untuk kegiatan harian.      | • Melihat dokumen SK dan prosedur kerja.<br/>• Membuat daftar hadir.<br/>• Membuat dan mengelola pemendek URL. |
| **Laboratory Admin**        | Pengelola kegiatan laboratorium.                                     | • Menerima dan memverifikasi pendaftaran laboratorium.<br/>• Mengelola data pengiriman sampel (shipment registration).<br/>• Membuat laporan hasil pemeriksaan. |
| **Clinic Admin**            | Pengelola data pelayanan klinik.                                     | • Mengelola pendaftaran layanan klinik.<br/>• Memasukkan dan memperbarui data pasien serta pemeriksaan. |
| **Doctor**                  | Tenaga medis dalam pemeriksaan atau verifikasi hasil.                | • Melihat data pendaftaran klinik dan laboratorium.<br/>• Mengakses informasi hasil pemeriksaan. |
| **Scientific Tourism Admin**| Pengelola kegiatan wisata ilmiah dan edukatif.                       | • Mengelola pendaftaran peserta kegiatan ilmiah.<br/>• Memverifikasi dan memperbarui data kegiatan serta laporan peserta. |
